# store-app

I created an E-Commerce application\
Technologies used:\
Text Editor: Visual Studio Code\
Back-end: .NET 5\
Front-end: Angular 11, Ngx-Bootstrap, HTML5, SCSS\
Databases: SQLite, Redis for in memory storing via Docker\
Payments: Stripe API
